Neuro Science

Cloud Analysis As a Service

Neuroscience Computational Analysis As a Service (NeuroCAAS) is a new platform that addresses computational infrastructure as an issue independent of scientific data analysis. We provide all of the tools necessary for a library of modern data analysis to function without ANY configuration on the part of the end user. See our open source project and paper below, or dive in to one of the analyses that we have made available

Giothub Repo Link
Paper Link

All analyses on NeuroCAAS share the same structure. Users chose one or more primary datasets to analyze, and then select a configuration file that will be used to analyze that data. Configuration files can specify individual parameters or paths to other parameter storage objects for efficiency. Every NeuroCAAS analysis has two optional parameters you can specify in a configuration file. If you know you have especially large data, specify the size of your dataset in gigabytes under with a '__dataset_size__' parameter. If you know how long your analysis should take, specify that time in minutes with a '__duration__' parameter. This will also keep your costs down. Processing information will appear in the status box as analyses are initialized, and more detailed information from the command line can be read out later on. At the end of processing, results can be downloaded back to your local machine. All users have the same starting budget of $300. If you are told that you are over budget, contact neurocaas@gmail.com to request an increase.
